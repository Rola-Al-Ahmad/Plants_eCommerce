# Leafcraft: Plants eCommerce Platform 🌿

Leafcraft is a modern, responsive e-commerce platform built with React and Vite, designed for plant lovers and gardening enthusiasts. It offers a seamless shopping experience for browsing, learning about, and purchasing a wide variety of plants and gardening supplies.

## 🔗 Live Demo

👉 [View Deployed App](https://Rola-Al-Ahmad.github.io/Plants_eCommerce)

---

## Features

- **Shop**: Explore a diverse catalog of plants and gardening products
- **Shopping Cart**: Add, remove, and manage items in your cart
- **Wishlist**: Save your favorite plants for future reference
- **Blog**: Read articles on plant care and gardening tips
- **Responsive Design**: Optimized for all devices
- **Search & Filter**: Quickly find products with advanced search and filtering
- **Secure Checkout**: Streamlined and safe checkout process
- **Contact**: Reach out to customer support easily

- React 18
- Vite
- React Router DOM
- React Toastify
- Modern CSS with bootstrap CSS
- Context API for state management

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rola-Al-Ahmad/Plants_eCommerce.git
   cd Plants_eCommerce
   ```
2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```
3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. **Open your browser** and visit [http://localhost:5173](http://localhost:5173)
